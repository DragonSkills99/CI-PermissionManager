Check Permissions

getUsersWith

$this->permissions->getUsersWith($permission);

returns the userObj with the given permission. The permission parameter can be the ID or the slug.

getPermissions

$this->permissions->getPermissions();

returns an array of Permission Objects for further checking.

getPermissionsByLevel

$this->permissions->getPermissionsByLeve($level);

returns an array of Permission Objects which has the $level as level for further checking. $level can be a numeric value or the slug.

getUserPermissions

$this->permissions->getUserPermissions([$userid]);

returns an array of permissions which the currently logged in or the given user possesses. The userid is optional an has to be an numeric value if used.

hasPermission

$this->permissions->hasPermission($id, [$userpermissions], [$checked_permissions], [$inherit_permissions = true]);

returns an boolean if the user has this permissions. $id can be an numeric value or the slug. $userpermissions could contain an array of Permission Objects which the User possesses. $checked_permissions could be an array of already checked Permission Objects. $inherit_permissions could be an boolean value, which sets if the method should only check for the given permission (false) or for any permission granted by the level.

needPermission

$this->needPermission($id);

$id can be an numeric value or the slug of the permission. This method checks if the user has the permission and if not it stops the code execution and shows an error.

getInheriting

$this->permissions->getInheriting($permission, [$userid], [$result], [$permissions]);

returns an array of permission ids which are granted by the given $permission. $permission can be an numeric value or the slug. $userid could be the id of an user. $result is the resulting array of ids. $permissions is a Permissions Object given for better performance, should be left null.

getPermissionID

$this->permissions->getPermissionID($slug);

returns the numeric ID of the given $slug. $slug can be a numeric value or the slug.

existsPermission

$this->permissions->existsPermission($id);

checks if the given permission exists and returns a boolean value. $id can be a numeric value or the slug.

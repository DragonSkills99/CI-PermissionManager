Added Sites

keys

PermissionManager/keys

returns key images

installation

PermissionManager/installation

lets you choose your superuser. After you've set one you can do it with the users site for further users.

install

PermissionManager/install

creates the needed tables in the database.

users

PermissionManager/users

lists the users of your site. By clicking on an entry you'll be redirected to the user page.

userjs

PermissionManager/userjs

contains some javascript code for the user site.

user

PermissionManager/user/$id

allows you to edit the permission of the given user. $id should be the numeric id, the E-Mail-Address or the username of the selected user.

editpermission

PermissionManager/editpermission/$userid/$permission/$active

an interface which toogles the permissions. it is user by the user site by ajax calls.

edit

PermissionManager/edit/$permission

allows you to edit the level of a permission. $permission can be a numeric value or the slug.

enroll

PermissionManager/enroll

this page lists all permission you've access to. If you click on the entries it redirects you to the edit site.

add

PermissionManager/add

allows you to create new permission entries. After successfully creating an entry you'll be redirected to the enroll site.

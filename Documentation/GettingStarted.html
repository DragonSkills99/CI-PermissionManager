Getting Started

For using the users library you first have to import it. Just call:

$this->load->library("Permissions");

If your outside of an Controller replace call "$this->permissions = new Permissions();" additionally.

Note: Permissions checked with this object will only fit for the user which is currently logged in. To check other users permission you have to send the ID (has to be a numeric value) as parameter e.g. "new Permissions(1);"

Now you have access to the users library.

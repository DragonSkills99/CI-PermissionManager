Manage Permissions

setPermission

$this->permissions($permission, $userid, $state);

grants/takes the given user the given permission. $permission can be a numeric value or the slug. $userid has to be the id of the user. $state has to be a boolean value which grants (true) or takes (false) the permission.

addPermission

$this->permissions->addPermissions($name, $level_slug);

creates a new permission. $name has to be the name as a string. $level_slug has to be the slug of the upper permission. (or the slug of the currently creating permission)

updatePermissionLevel

$this->permissions->updatePermissionLevel($permission, $level);

set a new level for the given permission. $permission can be a numeric value or the slug. $level can be a numeric value or the slug.

deletePermission

$this->permissions->deletePermission($permission);

deletes the given permission. $permission can be a numeric value or the slug.

Warning: You should not delete the superuser permission. If you've done it you could reset it to your user by calling PermissionManager/installation in your browser.
